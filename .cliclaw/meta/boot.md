# Boot Instructions

## On First Run

- Confirm current branch is `matic-feature`. If not, create and switch to it: `git checkout -b matic-feature`
- Run `npm test` to verify baseline is green before touching anything
- Review `docs/` and `AGENTS.md` for context on what OpenClaw does and where CLIClaw can go further
- Check `git log --oneline -10` for recent work
- Never run `git push` — all work stays local for owner review
